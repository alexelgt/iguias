Guide

Save battery

Reduce the brightness and deactivate automatic brightness.

Change: go to Settings, Brightness.

Close applications you are not using.

Use only notifications of social applications or applications that are important.

Deactivate: go to Settings, Notifications.

Deactivate the location of the applications and the location services that you do not use frequently.

Deactivate: go to Settings, General, Location Services.

Use the Bluetooth only when needed.

Deactivate: go to Settings, General, Bluetooth.

Use the 3G connection only when needed.

Deactivate: go to Settings, General, Network.

Select the carrier you use.

Change: go to Settings, Carrier.
