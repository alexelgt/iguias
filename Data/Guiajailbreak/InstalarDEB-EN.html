Install DEB files

To install DEB files from the device, you need to have installed the application iFile (available in Cydia). This application is a paid application but you can use a free version.

After downloading a file, place it on the path you want. If you have downloaded it from an external app and you have opened the file in iFile it will be in the route /var/mobile/Documents.

Now press the DEB file and press "Install" and it will begin installing.
