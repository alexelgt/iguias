External memory

Introduction:

The iPad allows the use of some flash drives and external memories to import photos through the Camera Connection Kit but if we have iFile we can access all the files of the flash drive or external memory.

With this tutorial we'll be able to intefrate the management of our flash drive or external memory with some file managers or applications to play video formats not supported by the native video application.

Requirements:

  • Jailbreak.
  • Camera Connection Kit.
  • Compatible flash drive or external memory.
  • iFile (available in Cydia).

    • Steps to do:

    1. Connect the flash drive or external memory to the iPad with the Camera Connection Kit.

    2. Open iFile and go to the path /var/mnt.

    3. Press Edit, select the folder mount1, then press the lower right button and press Copiar/Enlazar.

    4. Go to the Documents folder of the application you wish to integrate the flash drive or external memory.

    5. Press Edit, then press the lower right button and press Crear enlace.

    6. Press on the blue arrow of the file and change its name (optional step).

    App that have been checked to be compatible:

    GoodReader
    OPlayerHD (free and paid version)
    PDF Expert
    ReaddleDocs
