Jailbreak

System modificaion that removes some limitations of the device. It allows access the entire file system of the device and install software not authorized by Apple.

Tethered jailbreak:

This kind of jailbreak requires a computer to reboot the device.

Semi-(un)tethered jailbreak:

This type of jailbreak is like a tethered jailbreak but the device can be rebooted having only some functions available.
For all functions to be available, it is necessary to reboot the device with a computer.

Untethered jailbreak:

This kind of jailbreak does not require a computer to reboot the device.
