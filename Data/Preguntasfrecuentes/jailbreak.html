Jailbreak

Modificación del sistema con la que se eliminan algunas limitaciones del dispositivo. Permite acceder a todo el sistema de archivos del dispositivo e instalar software no autorizado por Apple.

Jailbreak tethered:

Este tipo de jailbreak requiere de un ordenador para poder reiniciar el dispositivo.

Jailbreak semi-(un)tethered:

Este tipo de jailbreak es parecido a un jailbreak tethered pero el dispositivo puede reiniciarse teniendo solo disponibles algunas funciones.
Para que estén todas las funciones disponibles, es necesario reiniciar el dispositivo con un ordenador.

Jailbreak untethered:

Este tipo de jailbreak no requiere de un ordenador para poder reiniciar el dispositivo.
